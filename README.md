# 桃下村塾 ／ TOUKA SONJUKU

岡山から、社会を変革するリーダーを育てる学生団体「桃下村塾」の公式Webサイト。

## 概要

- AIとビジネスを学ぶ実践型の学びの場
- 岡山大学を中心とした学生団体（2025年〜）
- 中期目標：700人規模のホールイベント開催

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- フォント: Noto Serif JP, Cormorant Garamond
- ホスティング: Render (Static Site)

## ローカルでの確認

このフォルダで簡易サーバを起動すればOK:

```bash
py -m http.server 5173
```

ブラウザで `http://localhost:5173/` を開く。
